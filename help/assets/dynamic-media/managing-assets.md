# Administración de activos de Dynamic Media {#managing-assets}

La administración de recursos de Dynamic Media (imágenes, vídeos y recursos interactivos) una vez cargados implica muchas tareas, como previsualizar, descargar o publicar.

Al obtener una vista previa, también puede aplicar ajustes preestablecidos de imagen (solo para imágenes) o ajustes preestablecidos de visor (para vídeo e imágenes). No puede aplicar ambos a un recurso. Consulte [Aplicar ajustes preestablecidos de visor](viewer-presets.md) y [Aplicar ajustes preestablecidos de imagen](image-presets.md).

La información general sobre la administración de recursos se encuentra en [Administrar recursos con la interfaz de usuario táctil](/help/assets/manage-digital-assets.md).

* [Previsualización de recursos de Dynamic Media](previewing-assets.md)
* [Publicar recursos de Dynamic Media](publishing-dynamicmedia-assets.md)
