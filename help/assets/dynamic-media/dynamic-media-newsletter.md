# ![Logotipo de Dynamic Media Newsletter](/help/assets/dynamic-media/assets/dynamic-media-newsletter-logo.png)

Para ayudarle a obtener un valor de inmediato, esta newsletter está diseñada para ayudarle a ponerse al día rápidamente con Dynamic Media y Dynamic Media Classic.

Explore valiosos recursos de creación de conocimiento, como vídeos explicativos y descripciones generales de soluciones. Lea las introducciones a algunas de las funciones y funciones clave de la solución, como ajustes preestablecidos de imagen, ajustes preestablecidos de visor, perfiles de vídeo y mucho más.
