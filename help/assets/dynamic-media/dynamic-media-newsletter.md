# ![Logotipo de Dynamic Media Newsletter](/help/assets/dynamic-media/assets/dynamic-media-newsletter-logo.png)

Para ayudarle a obtener valor de inmediato, este boletín está diseñado para ayudarle a ponerse al día rápidamente con Dynamic Media y Dynamic Media Classic.

Explore valiosos recursos de creación de conocimientos, como vídeos explicativos y descripciones generales de soluciones. Lea las introducciones a algunas de las funciones y capacidades clave de la solución, como ajustes preestablecidos de imagen, ajustes preestablecidos de visor, perfiles de vídeo y mucho más.

## Insistirse. Manténgase informado.
