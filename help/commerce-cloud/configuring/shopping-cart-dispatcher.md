# Carro de compras y configuración de Dispatcher

[AEM componentes](https://github.com/adobe/aem-core-cif-components) principales de CIF viene con un carro de [compras](https://github.com/adobe/aem-core-cif-components/tree/master/ui.apps/src/main/content/jcr_root/apps/core/cif/components/commerce/minicart/v1/minicart) listo para usar que procesa un carro de compras y un formulario de cierre de compra. El carro de compras requiere un Dispatcher [AEM con la configuración del enrutamiento GraphQL](https://github.com/adobe/aem-core-cif-components/blob/master/dispatcher) . El siguiente vídeo detalla una configuración local y la configuración mediante una imagen de Docker.
