# Notas de la versión de Universal Editor Preview {#preview}

Estas son las notas de la versión de la **versión de vista previa** del editor universal. Estas características están disponibles actualmente en el **entorno de vista previa** del editor universal. Estas funciones están programadas para su lanzamiento general el 13 de noviembre de 2025.

Estas notas de la versión de **preview** se proporcionan para que sepa qué cambios se avecinan en el editor universal y pueda probarlos al [cambiar al entorno de vista previa.](/help/sites-cloud/authoring/universal-editor/navigation.md#user-properties)

>[!TIP]
>
>Para las **notas de la versión actuales** del editor universal, consulte el documento [Notas de la versión del editor universal.](/help/release-notes/universal-editor/current.md)

>[!NOTE]
>
>El contenido de la versión real y la fecha de lanzamiento están sujetos a cambios.

## Próximas nuevas funciones {#what-is-new}

* Se han mejorado el menú contextual en el encabezado del panel y el carril derecho.
* los esquemas json ahora se entregan para su validación.
* El inicio de sesión en AEM para desarrolladores es ahora una parte fundamental del editor universal y ya no se necesita la extensión.

## Próximas mejoras {#other-improvements}

* Se ha ajustado la forma en que el árbol de estructura se expande automáticamente de forma predeterminada.
* Se ha añadido una anchura específica para el para el menú contextual.
* Se corrigieron errores del servicio de editor universal debido a ciertos valores de `null`.
