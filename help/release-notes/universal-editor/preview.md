# Notas de la versión de Universal Editor Preview {#preview}

Estas son las notas de la versión de la **versión de vista previa** del editor universal. Estas características están disponibles actualmente en el **entorno de vista previa** del editor universal. Estas funciones están programadas para su lanzamiento general el 21 de mayo de 2026.
