# Notas de la versión 2025.11.13 del editor universal {#release-notes}

Estas son las notas de la versión del editor universal del 13 de noviembre de 2025.

>[!TIP]
>
>Si desea probar las **próximas** funciones del editor universal antes de su lanzamiento, consulte las [notas de la versión del editor universal.](/help/release-notes/universal-editor/preview.md)

>[!TIP]
>
>Consulte [esta página](/help/release-notes/release-notes-cloud/release-notes-current.md) para ver las notas de la versión actuales de Adobe Experience Manager as a Cloud Service.

## Novedades {#what-is-new}

* Las acciones contextuales se han trasladado a menús desplegables.
* los esquemas json ahora se entregan para su validación.
* [El inicio de sesión de desarrollador local](/help/sites-cloud/authoring/universal-editor/navigation.md#local-developer-login) es ahora una parte fundamental del Editor universal y ya no se necesita la extensión.
* El carril derecho se ha migrado a una nueva interfaz de usuario.
* Los elementos del árbol de contenido ahora se expanden automáticamente.

## Funciones de adopción anticipada {#early-adopter}

Si le interesa probar estas nuevas funciones y compartir sus comentarios, envíe un correo electrónico a su Adobe Customer Success Manager desde la dirección de correo electrónico asociada a su Adobe ID.

### Nuevo RTE {#new-rte}

El nuevo RTE de ProseMirror, que cuenta con un selector de páginas en el cuadro de diálogo de vínculos, ya está disponible en el panel derecho. [Este RTE cuenta con opciones de configuración flexibles.](/help/implementing/universal-editor/configure-rte.md)

## Otras mejoras {#other-improvements}

* Se ha añadido una anchura específica para el para el menú contextual.
* Se corrigieron errores del servicio de editor universal debido a ciertos valores de `null`.
